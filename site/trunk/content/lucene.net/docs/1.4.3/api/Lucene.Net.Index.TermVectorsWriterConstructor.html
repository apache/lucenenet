TermVectorsWriter Constructor 

Create term vectors writer for the specified segment in specified directory. A new TermVectorsWriter should be created for each segment. The parameter 

maxFields
indicates how many total fields are found in this document. Not all of these fields may require termvectors to be stored, so the number of calls to 
openField
is less or equal to this number.
